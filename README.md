# HealthTrack-Android

HealthTrack-Android is a **mobile healthcare application** built for Android devices. The app aims to help users monitor their health, track fitness activities, and maintain overall wellness in a simple and intuitive way.

## Features

- Track daily health metrics like steps, heart rate, and calories burned.
- Log meals and water intake.
- Set reminders for medications and workouts.
- Visualize progress with charts and statistics.
- User-friendly interface optimized for Android devices.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/aaditya-gurav/HealthTrack-Android.git


2. Unzip the project folder (if downloaded as a zip).
3. Open the project in **Android Studio**.
4. Let Gradle sync and build the project.
5. Run the app on an emulator or a physical Android device.

## How to Run the App

1. Open **Android Studio** and load the project.
2. Connect an Android device or start an emulator.
3. Click **Run** (green play button) in Android Studio.
4. Explore the app’s features.
```
